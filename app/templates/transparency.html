Transparency

I want to be as open as possible about how xygt is run, and what we actually do.

See our log of requests for information here.

xygt.cc runs on a Hetzner VPS located in Nuremberg, Germany.

The specs of this machine are as follows:

  • 2 Core ARM vCPU
  • 4GB RAM
  • 80GB SSD Storage across 2 volumes
  • 1 IPv4 Address

xygt.cc is ran by a single person, and is not affiliated with any other service.

The backend for this platform was written using Python with the Flask framework.

Our database is self-hosted on this server with no internet access, it runs MongoDB Community Edition and stores user data, file location data, and URL redirect data.

Any extra info you'd like to see on this page can be suggested through the E-Mail on the contact page
